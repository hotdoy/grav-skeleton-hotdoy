## Reveal.js 

Reveal content with [Reveal.js](https://github.com/hotdoy/reveal.js).
A a crazy easy to use JavaScript library to easily animate elements as they enter the viewport.
It uses the intersection observer API instead of scroll events so it's blazing fast.
